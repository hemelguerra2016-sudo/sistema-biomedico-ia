
# Sistema IA de Preorientación Biomédica

## Descripción

Aplicación web desarrollada con Streamlit para apoyo preliminar en clasificación básica de síntomas clínicos.

El sistema utiliza:
- reglas biomédicas simples
- selección estructurada de síntomas
- IA generativa para orientación educativa

## Objetivo

Apoyar la organización preliminar de síntomas y orientar al usuario sobre posibles niveles de atención clínica.

## Características

- Clasificación clínica básica:
  - Verde
  - Amarillo
  - Rojo

- Selección estructurada de síntomas

- Campo de texto adicional

- Generación de orientación biomédica educativa usando IA

## Tecnologías utilizadas

- Python
- Streamlit
- Gemini API
- Google Colab
- GitHub

## Limitaciones

- No realiza diagnósticos médicos
- No prescribe tratamientos
- No reemplaza profesionales de salud
- Requiere validación clínica real

## Aspectos éticos

El sistema fue diseñado considerando:
- privacidad de datos
- limitación de decisiones automáticas
- interpretabilidad
- reducción de riesgo clínico
